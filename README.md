# GenZSlangConvertor
Gen-Z Slang Translator is a beginner-friendly Java project that converts normal English sentences into Gen-Z slang using OOP concepts, collections, randomness, and multithreading, with multiple tone modes (Soft, Savage, Unhinged) for dynamic, fun translations.

## Features
- Three tone modes: Soft, Savage, Unhinged
- Uses interfaces, abstract classes, inheritance
- Random slang generation every run
- Case-preserving word replacement
- Multithreaded translation
- Runs in a loop until exit

## How to Run
```bash
javac GenZSlangTranslator.java
java GenZSlangTranslator
