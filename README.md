# GenZSlangConvertor
Gen-Z Slang Translator is a Java project that converts normal English sentences into Gen-Z slang using OOP concepts, collections, randomness, and multithreading, with multiple tone modes (Soft, Savage, Unhinged) for dynamic, fun translations.

## Features
- Generates GenZ Sentence in 3 tone modes: Soft, Savage, Unhinged
- Uses interfaces, abstract classes, inheritance
- Random slang generation every run
- Case-preserving word replacement
- Multithreaded translation
- Runs in a loop until exit

## Core Java Concepts Used

-Interfaces
-Abstract classes
-Inheritance
-Polymorphism
-HashMap and Collections
-String manipulation
-Randomization
-Multithreading
-User input handling

## How to Run
```bash
javac GenZSlangTranslator.java
